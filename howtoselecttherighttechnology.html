The technologies used in website building form the web slack namely: 1. Client side coding called the html, css and Java script 2. Programming lanaguage called the php and the microsoft ASP.net 3. Database Engine called MySQL and microsoft MySQL server 4. Web Server Software called Apache and IIS 5. Operating system such as the window How to select the kind of technology for the website 1. First and foremost, the technology we choose should depend on the problem we want to solve. Soem products are better done in Phyton than in Java. 2. Think of who will be using the product and make it user friendly. 3. Think of where the product will be used on frequently: tablet or mobile. 4. Take into consideration the browser that will be used most often. 5. Time should be taken into consideration and either build from scratch or use an open-source technology. 6. The ecosytem behind the language and tools we intend to use in building the website should be taken into consideration. 7. A mature technology that has a guarantee to lasting over a long period of time should be preferred over a technology of no guarantee of lasting over a long period of time.
